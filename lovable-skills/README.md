## How to Install

### Option 1: Import from ZIP (Recommended)
1. Open Lovable.dev → Settings → Skills
2. Click **Add → Upload ZIP**
3. Drag in this ZIP file
4. Lovable will validate and publish the skills to your workspace

### Option 2: Manual (One Skill at a Time)
1. Extract the ZIP
2. For each skill folder in `skills/`, zip just that folder
3. Upload each skill ZIP individually via Settings → Skills → Add → Upload ZIP

### Option 3: Import from GitHub
1. Push this repo to GitHub
2. In Lovable.dev → Settings → Skills → Add → Import from GitHub
3. Paste the repo URL

## Skill Format

Each skill follows the [official Lovable.dev SKILL.md format](https://docs.lovable.dev/features/skills),
which is also compatible with Anthropic's Claude Agent Skills convention.

```
skill-folder/
├── SKILL.md          # Required: name, description, instructions
└── reference.md      # Optional: bundled reference files
```

### SKILL.md Anatomy
```markdown
---
name: skill-name
description: Use when [trigger]. [Scope]. [Boundary].
---

# Skill Title

[Instructions the AI follows when this skill is loaded]
```

# Landing Section: Industry section

Industry section — tailored for X industry.

## Purpose
- What this section accomplishes
- Where in the page it goes
- What it should make the visitor do

## Anatomy
- Layout structure
- Elements and their roles
- Visual hierarchy

## Copy Guidelines
- Headline patterns
- Subhead patterns
- CTA text
- Microcopy

## Design Patterns
- Layout (grid, columns, alignment)
- Spacing (generous, breathing room)
- Color (use brand + accent)
- Typography (clear hierarchy)
- Imagery (real > stock, people > objects)
- Motion (subtle, purposeful)

## Mobile Considerations
- Stack columns
- Reduce padding
- Larger touch targets
- Hide non-essential elements

## Accessibility
- Semantic HTML
- ARIA where needed
- Keyboard navigable
- Color contrast 4.5:1
- Don't rely on color alone

## Conversion Best Practices
- One primary CTA
- Reduce friction
- Social proof
- Urgency / scarcity (only if real)
- Clear value prop
- Above the fold (for hero)

## A/B Test Ideas
- Headline variations
- CTA text
- CTA color
- Image vs video
- With / without social proof

## Implementation
Provide React + Tailwind code:
- Responsive
- Accessible
- Animated on scroll (subtle)
- Performance-optimized

## Forbidden Patterns
- Multiple competing CTAs
- Vague headlines
- Stock photos of handshakes
- Walls of text
- Auto-playing video with sound
- Pop-ups covering content

## Output Format
1. Section code (React + Tailwind)
2. Copy template
3. Mobile layout notes
4. A/B test variants

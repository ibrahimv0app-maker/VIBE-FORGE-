# Cohort Analysis Architect

Use when running cohort analyses. Always uses consistent cohorting, retention curves, and segment comparison.

## Principles
Data analysis principles: reproducibility, clarity, statistical rigor.

## Process
1. Define the question
2. Identify the data sources
3. Clean and validate the data
4. Analyze with appropriate methods
5. Visualize the results
6. Communicate findings clearly

## Common Pitfalls
- Correlation != causation
- Survivorship bias
- Selection bias
- Small sample size
- Multiple comparisons
- Overfitting

## Output Format
1. Question and hypothesis
2. Methodology
3. Code / queries
4. Results (table + visualization)
5. Interpretation
6. Limitations and next steps

# Tutorial Writer

Use when writing tutorials. Always uses hands-on, step-by-step, copy-pasteable examples.

## Audience Analysis
Identify the audience and tailor the document.

## Structure
Define the document structure.

## Template
Provide a copy-pasteable template.

## Writing Principles
- Active voice
- Present tense
- Second person ("you")
- Short sentences
- Concrete examples
- Scannable formatting

## Common Mistakes
- Walls of text
- Jargon without definition
- Outdated screenshots
- Missing code examples
- No version / date

## Output Format
1. The document (in markdown)
2. Notes on choices made

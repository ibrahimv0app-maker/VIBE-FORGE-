# Build a CRM platform

contacts, deals, pipelines, activities, automation

- Frontend (web, mobile, or both)
- Backend (API, services, workers)
- Database (primary, cache, search)
- Third-party services
- Infrastructure

## Data Model
Define the core entities and relationships.

## User Flows
Document the critical user flows:
- Onboarding
- Core activity
- Account management
- Billing
- Support

## Industry Compliance
Document applicable regulations and how to comply.

## Security
Industry-specific security concerns and mitigations.

## Performance
Industry-specific performance considerations.

## Scaling
How the architecture scales with users / data.

## Monetization
Common business models and how to implement them.
